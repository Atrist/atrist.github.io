# 1. Setting Sync

使用 github gits 作为文件,来云上保存 vscode 的插件目录,可以用于多终端同步你的 vscode 插件

前提条件: 需要一个 github 账号

# 快捷键

  1. 上传 : Shift + Alt + U
  2. 下载 : Shift + Alt + D

(on macOS: Shift + Option + U / Shift + Option + D)

# 2. Draw.io Integration

用来在 vscode 中画流程图

# 3. markdown 相关的插件

# 1. Github Markdown Preview

# 2. markdown TOC

通过 # ## ### 等标题生成对应标题目录链接

# 4. Vetur

# 5. open in browser

# 6. AUto Rename Tag

# 7. Material ICon Theme
